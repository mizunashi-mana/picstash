# Picstash プロダクト概要

## 目的

Picstash は、イラストや画像作品を効率的に整理・管理するための Web アプリケーションです。AI による自動タグ付けや類似画像検出により、**手作業をほとんど必要とせず**大量の画像コレクションを管理できます。

## ターゲット

- 個人利用（シングルユーザー、ログイン不要）
- 中規模コレクション（数万枚程度）
- レスポンシブ対応（PC / タブレット / スマホ）

## コア機能

### 1. 画像管理
- Web UI からのドラッグ＆ドロップでアップロード
- サムネイル付きギャラリー表示
- 画像の詳細表示・フルスクリーン表示
- 画像の削除

### 2. AI 自動タグ付け
- ローカル AI モデル（CLIP 等）による画像解析
- アップロード時に自動でタグを推薦・付与
- キャラクター、背景、色調、構図などを自動認識
- 必要に応じて手動でタグの追加・編集も可能

### 3. 類似画像検出
- **重複検出**: 同一または微修正版の画像を検出して整理
- **スタイル類似**: 画風や色調が似た作品を自動グルーピング
- 類似画像の一覧表示

### 4. 検索機能
- タグによる絞り込み検索
- キーワード検索
- 複数条件での検索（AND/OR）

### 5. コレクション機能
- シリーズもの（ストーリー、漫画など）の画像をまとめて管理
- 手動でコレクションを作成・編集・削除
- 作品をコレクションに追加・削除・並び替え
- コレクション単位での連続閲覧（ページ送り）

## ユースケース

### UC-1: イラストのアップロード
1. ユーザーが画像をドラッグ＆ドロップでアップロード
2. AI が自動で画像を解析しタグを付与
3. ユーザーは必要に応じてタグを確認・修正
4. ギャラリーに追加される

### UC-2: 類似画像の整理
1. システムが自動で類似・重複画像を検出
2. ユーザーが類似画像一覧を確認
3. 不要な重複を削除、または類似作品をまとめて閲覧

### UC-3: 作品の検索と閲覧
1. ユーザーがタグまたはキーワードで検索
2. 検索結果がギャラリー形式で表示される
3. 気になる作品をクリックして詳細表示

### UC-4: シリーズ作品の管理
1. ユーザーがシリーズ用のコレクションを作成（例: 「○○漫画」）
2. シリーズの画像をコレクションに追加し、順番を並び替え
3. コレクションを開いてページ送りで連続閲覧

## 技術方針

### AI/ML
- **画像埋め込み**: CLIP モデルでベクトル化し、類似検索・タグ推薦に活用
- **ローカル実行**: 外部 API 不要、プライバシー重視

### パフォーマンス
- 中規模（数万枚）でも快適に動作
- サムネイル生成による表示高速化
- ベクトル検索の効率化
