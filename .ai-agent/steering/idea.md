# アイデア

現状のプロダクトに対する改善アイデア置き場。雑多にアイデアを書き、プランニングのためにどういうアイデアなのかを分類してより解像度を上げて整理しておく。

## 不具合

明らかに改善の必要がある・仕様に反している事象。

| 内容 | 温度感 |
|------|--------|
| 統計表示で画像サムネイルが表示されない問題 | 中 |
| 統計表示でグラフの表示が崩れている問題 | 中 |
| 特定の ZIP ファイルで、「End of Central Directory Record not found」というエラーが起きて取り込みができない | 中 |

## 機能要望

検討の価値のある追加機能・既存機能の改善アイデア。

| 内容 | 温度感 |
|------|--------|
| ギャラリーのカルーセル表示 | 中 |
| 閲覧・編集モード分離 | 中 |
| client / server での型の共通化 | 中 |
| 属性の発音埋め込みによる、検索性の向上 | 中 |
| 検索バーのリファクタリングと安定性向上 | 中 |
| ジョブワーカーのグレースフルシャットダウン | 中 |
| 非同期ジョブ管理機能（ジョブ完了時の通知と続きの操作） | 中 |
| 設定読み込みをグローバル変数でやるのをやめる | 低 |
| client パッケージを web-client パッケージに改名 | 低 |
| replan コマンド作成 | 低 |
| 複数サーバ対応 | 低 |
| 認証対応 | 低 |
| 多言語対応 | 低 |
| ヘルプページ作成。チュートリアル作成 | 低 |
| アップロードのリアルタイム進捗表示 | 低 |
| URL 属性のリンク表示 | 低 |
| テストカバレッジ閾値のCI確認 | 低 |
